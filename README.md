## 📌 Featured Projects

### 🏭 Warehouse Optimiser
> A full-stack warehouse-operations platform where computer vision, OCR, and an LLM run as a **multi-stage agentic workflow** (YOLOv8 → EasyOCR → Groq LLM) to automate inventory ingestion and product verification. Backend on **FastAPI + async SQLAlchemy + PostgreSQL** with REST & WebSocket APIs, **ARIMA-based forecasting**, and a recommendation engine for shipment prioritisation. React + Vite dashboard with live camera capture and one-click Excel exports, fully containerised with Docker Compose.

`Python` · `FastAPI` · `YOLOv8` · `EasyOCR` · `Groq LLM` · `ARIMA` · `WebSocket` · `Docker`

🔗 [View Repository](https://github.com/Jaskirat314276/WAREHOUSE-OPTIMIZATION)

Prompt-engineering and retrieval workflows improve content quality and factual grounding, served through a configurable Streamlit interface supporting multiple topics and languages.
