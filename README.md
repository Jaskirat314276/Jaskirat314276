## 🚀 Featured Projects

### 🏭 &nbsp;Warehouse Optimiser
A full-stack warehouse-operations platform where computer vision, OCR, and an LLM run as a **multi-stage agentic workflow** (YOLOv8 → EasyOCR → Groq LLM) to automate inventory ingestion and product verification. Backed by **FastAPI + async SQLAlchemy + PostgreSQL** with REST & WebSocket APIs, **ARIMA forecasting**, and a recommendation engine for shipment prioritisation — plus a React + Vite dashboard with live camera capture and one-click Excel exports, fully containerised with Docker Compose.
